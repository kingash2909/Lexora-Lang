## 🚨 TROUBLESHOOTING

### Common Issues

**Issue: "fatal: remote origin already exists"**
```bash
# Solution: Remove and re-add
git remote remove origin
git remote add origin https://github.com/YOUR_USERNAME/lexora-lang.git
```

**Issue: "Permission denied (publickey)"**
```bash
# Solution: Use HTTPS instead of SSH
git remote set-url origin https://github.com/YOUR_USERNAME/lexora-lang.git
```

**Issue: "Updates were rejected because the remote contains work"**
```bash
# Solution: Force push (careful!)
git push -f origin main
```

**Issue: Large files won't upload**
```bash
# Solution: Check .gitignore is working
git status
# Remove any large files that shouldn't be there
git rm --cached filename
git commit -m "Remove large files"
```
