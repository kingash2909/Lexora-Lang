About Lexora

Revolutionizing programming through natural language

🚀 What is Lexora?

Lexora 1.0 Ultimate Edition is a revolutionary programming language that bridges the gap between human language and machine execution. It allows you to write code in plain English, making programming accessible to everyone.

100% English Syntax
0 Syntax Barriers
∞ Possibilities
1.0 Current Version

Our Mission

To make programming as natural as human conversation, removing barriers and empowering anyone to create software.

📖 Language Philosophy

  • Natural First: Code should read like human language
  • Accessibility: Programming for everyone, not just experts
  • Power & Simplicity: Advanced features without complexity
  • Education Focused: Perfect for learning programming concepts

⚡ Core Features

Natural Language Syntax

Write code using English keywords like Display, Set, If, Repeat, Define, etc.

Display "Hello, World!"
Set name to "Developer"
Display "Welcome, ", name

Object-Oriented Programming

Full support for Classes, Inheritance, Interfaces, and Abstract methods

Class Animal:
    Define speak as:
        Display "Sound"
    End
End

Class Dog inherits Animal
End

Error Handling

Production-ready error management with Try-Catch-Finally blocks

Try:
    Write "data.txt", "Hello"
Catch:
    Display last_error
End

File & Network I/O

Read/write files, make HTTP requests, handle data

Read "input.txt"
Write "output.txt", result
HTTP GET "https://api.example.com"

Concurrency & Async

Multi-threading, locks, and async operations

Create Lock "mylock"
Acquire "mylock"
# Critical section
Release "mylock"

Web-Based IDE

Beautiful, modern editor with syntax highlighting and instant execution

📥 Installation Guide

Choose your platform and follow the simple steps below

Install via pip (Recommended)

1

Install Python

Make sure Python 3.8+ is installed

python --version
2

Install Lexora

One command installation

pip install lexora-lang
3

Run the Editor

Start the web-based IDE

lexora-editor

Browser will open automatically at http://localhost:5001

macOS Standalone Application

Completely independent - no Python or Homebrew needed!

🍎 Method 1: DMG Installer (Recommended)

  1. Download Lexora-1.0.0-macOS.dmg
  2. Open the DMG file
  3. Drag Lexora.app to Applications folder
  4. Open Terminal
  5. Add to PATH (one-time setup):
# Add this to your ~/.zshrc or ~/.bash_profile: export PATH="/Applications/Lexora.app/Contents/MacOS:$PATH" # Then reload: source ~/.zshrc # Now you can use: lexora hello.lx lexora-editor

📦 Method 2: PKG Package

  1. Download Lexora-1.0.0.pkg
  2. Double-click to install
  3. Installs to /usr/local/bin automatically
  4. Ready to use in any terminal

System Requirements

  • macOS 10.13 (High Sierra) or later
  • Intel or Apple Silicon (M1/M2/M3)
  • No Python or Homebrew required!
  • 512 MB RAM
  • 100 MB disk space

Quick Start After Installation

# Open Terminal and type: lexora --help # Run your first program: echo 'Display "Hello, World!"' > hello.lx lexora hello.lx # Launch web editor: lexora-editor # Or open Lexora.app from Applications

Windows Standalone Installer

No Python required! Our installer includes everything you need.

📦 Method 1: EXE Installer (Recommended)

  1. Download Lexora-1.0.0-Windows.exe from GitHub Releases
  2. Double-click to run installer
  3. Click "Next" through the wizard
  4. Installer adds Lexora to your PATH automatically
  5. Open Command Prompt or PowerShell
  6. Type: lexora
lexora hello.lx

💾 Method 2: Portable ZIP

  1. Download Lexora-1.0.0-Windows.zip
  2. Extract to any folder (e.g., C:\Program Files\Lexora)
  3. Add folder to system PATH (optional)
  4. Run lexora.bat from the folder

System Requirements

  • Windows 10 or later (64-bit)
  • No Python installation needed!
  • 512 MB RAM
  • 100 MB disk space
  • .NET Framework 4.7+ (usually pre-installed)

Quick Start After Installation

REM Open Command Prompt and type: lexora --help REM Run your first program: echo Display "Hello, World!" > hello.lx lexora hello.lx REM Launch web editor: lexora-editor

Linux Standalone Binary

Works on any distribution - no package manager needed!

🐧 Method 1: DEB Package (Debian/Ubuntu)

  1. Download lexora_1.0.0_amd64.deb
  2. Install with one command:
sudo dpkg -i lexora_1.0.0_amd64.deb
  1. Ready to use system-wide!
lexora hello.lx lexora-editor

📦 Method 2: Universal Binary (All Distros)

  1. Download Lexora-1.0.0-Linux.tar.gz
  2. Extract to /opt:
sudo tar -xzf Lexora-1.0.0-Linux.tar.gz -C /opt/
  1. Create symlinks:
sudo ln -s /opt/lexora/lexora /usr/local/bin/lexora sudo ln -s /opt/lexora/lexora-editor /usr/local/bin/lexora-editor
  1. Use anywhere!

⚡ Method 3: AppImage (Coming Soon)

  1. Download Lexora-1.0.0-x86_64.AppImage
  2. Make executable:
chmod +x Lexora-1.0.0-x86_64.AppImage ./Lexora-1.0.0-x86_64.AppImage

System Requirements

  • Any modern Linux distribution
  • Ubuntu 18.04+, Debian 10+, Fedora 30+, Arch, etc.
  • No Python or dependencies required!
  • glibc 2.17 or later
  • 512 MB RAM
  • 100 MB disk space

Build from Source

For developers who want to contribute or customize

🔧 Prerequisites

  • Python 3.8+ (for building only)
  • Git
  • PyInstaller (for building standalone binaries)

📥 Clone Repository

git clone https://github.com/yourusername/lexora.git cd lexora

🏗️ Build Standalone Binaries

# Install build dependency pip install pyinstaller # Build standalone executables python build_standalone.py

This creates platform-specific installers in the dist/ folder that don't require Python!

🧪 Test Without Building

# For development/testing only python lexora-web-editor/src/lexora/lexora.py hello.lx python lexora-web-editor/src/app.py
