# Lexora Web Editor

## Overview
Lexora Web Editor is a web-based code editor that enhances the features of the existing Simple English Interpreter. This project allows users to write, edit, and execute commands using a user-friendly interface.

## Features
- Code editing capabilities for the Simple English Interpreter.
- Real-time execution of commands.
- User-friendly interface with syntax highlighting.
- Ability to save and load code files.

## Installation
1. Clone the repository:
   ```
   git clone https://github.com/yourusername/lexora-web-editor.git
   ```
2. Navigate to the project directory:
   ```
   cd lexora-web-editor
   ```
3. Install the required dependencies:
   ```
   pip install -r requirements.txt
   ```

## Usage
1. Run the Flask application:
   ```
   python src/app.py
   ```
2. Open your web browser and go to `http://127.0.0.1:5000` to access the Lexora Web Editor.
